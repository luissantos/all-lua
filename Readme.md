# All lua versions

This project aims to provide docker containers with lua and its derivatives installed.

Current supported versions:

* Lua 5.1 + LuaRocks
* Lua 5.2 + LuaRocks
* Lua 5.3 + LuaRocks
* Lua 5.4 (preview)
* LuaJit 2.0 + LuaRocks
* LuaJit 2.1 (preview)
* Moonscript 0.5 + Lua 5.2 + LuaRocks

All containers are based on alpine linux.
